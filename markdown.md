# Markdownの記述

## ヘッダー

- 行の先頭に#をつけることでH1～H6まで作成できる
- #の数で文字サイズを調整可能

## コードブロック

- 記事内に[```]を入れることでコードブロックを挿入できる。また、先頭の記号の後ろに言語名を入れることでシンタックスハイライトが適用される。

    </br>

    (例)centosにてvimでcrontabを開くコマンド

    ```centos
    vim /etc/crontab
    ```

## テーブル

- Markdownテーブルの基本記法。以下は表題ありの３行３列テーブルの例

```markdown
| 列名１     | 列名２     | 列名３     | 
| ---------- | ---------- | ---------- | 
| １行１列目 | １行２列目 | １行３列目 | 
| ２行１列目 | ２行２列目 | ２行３列目 | 
| ３行１列目 | ３行２列目 | ３行３列目 | 
```

- プレビュー
  
| 列名１     | 列名２     | 列名３     |
| ---------- | ---------- | ---------- |
| １行１列目 | １行２列目 | １行３列目 |
| ２行１列目 | ２行２列目 | ２行３列目 |
| ３行１列目 | ３行２列目 | ３行３列目 |

</br>
