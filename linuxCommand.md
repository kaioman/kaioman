# Linuxコマンド

- 1.ディレクトリ削除

  - -fオプション→確認なしで削除する
  - -rオプション→再帰的に削除する
  - ディレクトリ内にファイルがあっても削除する
  
    ```sh
    rm -rf ディレクトリ名
    ```
  
  - 削除時に確認を行う場合(yで許可,nでキャンセル)

    ```sh
    rm -r ディレクトリ名
    ```

  - 以下のコマンドではディレクトリ内にファイルがあるとエラーとなる

    ```sh
    rmdir ディレクトリ名
    ```

- 2.ファイルコピー
  
  - 通常のコピー(基本形)

    ```sh
    cp コピー元 コピー先
    ```
